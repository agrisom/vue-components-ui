# ðŸ’¬ How to use Tooltips

The UITooltip component uses the vue [teleport](https://vuejs.org/api/built-in-components.html#teleport) function to show the component content in another layer to avoid overflow problems that may be generated by using it in other container elements.

## Use

In a layout component, we must add the UITooltipZone component, which will act as a container for the tooltip content. This applies the preloaded styles with 'style.css' to occupy the entire screen without responding to mouse events.

**App.vue**

```html
<script>
import { UITooltipZone } from 'vue-components-ui';
</script>

<template>
  <UITooltipZone />
</template>
```
