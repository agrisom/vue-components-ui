The UIMenu component uses the vue [teleport](https://vuejs.org/api/built-in-components.html#teleport) function to show the dropdown content in another layer to avoid overflow problems that may be generated by using it in other container elements.

## Use

In a layout component, we must add the UIMenuZone component, which will act as a container for the menu content. This applies the preloaded styles with 'style.css' so that it occupies the entire screen without responding to mouse events.

**App.vue**

```html
<script>
import { UIMenuZone } from 'vue-components-ui';
</script>

<template>
  <UIMenuZone />
</template>
```
